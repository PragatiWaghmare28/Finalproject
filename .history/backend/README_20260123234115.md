# Backend - Setup & Run

Quick notes to run the backend locally.

1) Copy environment example:

   cp backend/.env.example backend/.env

   Edit `backend/.env` and set `DATABASE_URL`, API keys, and `UPLOAD_DIR`.

2) Install (if not already):

   - Python 3.11 is required for full dependencies.
   - From project root:

     python3.11 -m pip install --user -r backend/requirements.txt

3) Run the backend:

   # from repo root
   backend/run.sh start

   Check logs: `tail -f backend/uvicorn.log`

   Health: http://127.0.0.1:8000/health

4) Stop:

   backend/run.sh stop

Notes:
- `backend/run.sh` uses `python3.11` and sets `PYTHONPATH` to repository root so `backend` is importable.
- For production use, prefer running inside Docker (see `docker-compose.yml`) or a process manager (systemd).
# Video-RAG Backend

FastAPI backend for Video-RAG application.

## Setup

```bash
# Create virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Run the server
uvicorn app.main:app --reload --port 8000
```
